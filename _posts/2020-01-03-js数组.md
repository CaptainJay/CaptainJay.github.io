## 一.数组
### 1.1 join()
  功能：将数组中所有元素都转化为字符串并连接在一起。  
  示例代码：[1,2,3,4].join('-')  
  示例结果："1-2-3-4"  
  ```
  function joinxxx(spe){
        let result = '';
        for (let value of this) {
          let dealValue = value?value:'';
          result = result==''? dealValue:result+(spe?spe:",")+dealValue;
        }
        console.log(result);
        return result;
  }
  Array.prototype.joinxxx = joinxxx;
  [1,2,3,4,5].joinxxx('-');
  ```
